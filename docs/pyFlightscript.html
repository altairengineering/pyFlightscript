pyFlightscript.actuators.edit_actuator(actuator, name, actuator_type, frame, axis, offset, radius, ct=None, rpm=None, swirl_velocity=None, velocity=None, density=None, cm=None)¶
Example usage:
edit_actuator(, 1, “Prop-1”, “PROPELLER”, 2, 1, 0.5,

1.2, ct=0.013, rpm=7000, swirl_velocity=”ENABLE”)

Appends lines to script state to edit an actuator.

Parameters:

  • actuator – Index of the actuator to be edited.

  • name – Name to be assigned to the selected actuator.

  • actuator_type – Type of the actuator (PROPELLER or JET_EXHAUST).

  • frame – Index of the coordinate system to be used for this actuator.

  • axis – Directional axis for the actuator disc.

  • offset – Offset distance along the axis for the actuator disc.

  • radius – Radius of the actuator disc.

  • ct – Thrust coefficient of the propeller (for PROPELLER type only).

  • rpm – RPM of the propeller (for PROPELLER type only).

  • swirl_velocity – ENABLE/DISABLE swirl velocity (for PROPELLER type only).

  • velocity – Exhaust velocity (for JET_EXHAUST type only).

  • density – Density of the exhaust jet (for JET_EXHAUST type only).

  • cm – Jet spreading coefficient (for JET_EXHAUST type only).

Appends lines to script state to create a series of automatic annular cross-sections.

Args:

Example usage: cad_create_auto_annular_cross_sections(1, 20, 2)

pyFlightscript.cad.cad_create_auto_cross_sections(frame=1, axis='Y', sections=20, body_index=1, growth_scheme=3, growth_rate=1.2, symmetry='NONE', cad_mesh='MESH')¶

Appends lines to script state to create a series of automatic cross-sections from mesh body or CAD body.

Parameters:

  • frame – Index of coordinate system to be used. Use value of 1 for reference coordinate system. Defaults to 1.

  • axis – Sweep direction for creating cross-section curves. One of the following: X, Y or Z. Defaults to ‘Y’.

  • sections – Number of cross-sections requested (> 1). Defaults to 20.

  • body_index – Index of the mesh body to be used for creating cross-sections (> 0). Defaults to 1.

  • growth_scheme – Clustering scheme to be used when positioning cross-sections along the sweep axis. One of the following: 1: Uniform, 2:Successive, 3:Dual-successive, 4: Reverse-successive. Defaults to 3.

  • growth_rate – Growth rate to be used with the selected growth_scheme. Use value of 1.0 for uniform spacing of cross-sections. Defaults to 1.2.

  • symmetry – Specify which symmetry plane (if any) to be used for creating half-section curves. Use one of the following: YZ, XZ or XY. If full section is desired instead, use NONE. Defaults to ‘NONE’.

  • cad_mesh – Specify whether the CAD body or mesh body with specified index should be used for generation of cross sections. Use ‘CAD’ for CAD body or ‘MESH’ for mesh body. Defaults to ‘MESH’.

Example usage:

pyfs.cad_create_auto_cross_sections(1, ‘Y’, 20, 1, 3, 1.2, ‘NONE’, ‘MESH’)
